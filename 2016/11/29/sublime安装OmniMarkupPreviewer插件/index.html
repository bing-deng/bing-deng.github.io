sublime安装OmniMarkupPreviewer插件

#sublime markdown 插件-OmniMarkupPreviewer插件

  • 安装OmniMarkupPreviewer插件后
  • cmd+option+o 运行404
    以下是解决办法

Sublime Text > Preferences > Package Settings > OmniMarkupPreviewer > Settings - User

  • 技巧：
    • mdl 加 tab 键 自动生成 [](link)
