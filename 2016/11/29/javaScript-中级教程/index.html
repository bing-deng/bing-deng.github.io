数字

1.JavaScript 不支持类，类这一概念在 JavaScript 通过对象原型（object prototype）得到延续（有关 ES6 类的内容参考这里Classes）
2.JavaScript 中的函数也是对象，JavaScript 允许函数在包含可执行代码的同时，能像其他对象一样被传递
3.JavaScript 还有一种内置Error（错误）类型
4.JavaScript 采用“IEEE 754 标准定义的双精度64位格式”（”double-precision 64-bit format IEEE 754 values”）表示数字,JavaScript 不区分整数值和浮点数值
5.内置对象 Math（数学对象），用以处理更多的高级数学函数和常数

闭包

JavaScript 中必须提到的功能最强大的抽象概念之一：闭包
与内部函数唯一不同的是：外部函数被返回了
一个闭包就是一个函数和被创建的函数中的作用域对象的组合
作用域对象组成了一个名为作用域链（scope chain）的链。它类似于原形（prototype）链一样，被 JavaScript 的对象系统使用。
